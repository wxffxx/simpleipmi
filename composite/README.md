# SI BMC Composite — 多主机管理系统

> 🚧 **开发中** — 占位目录

## 概述

基于 Ubuntu 的中央管理服务器，可统一管理多台 KVM Host 设备和视频采集卡，实现多目标机的远程控制。

## 核心功能规划

### 1. 设备发现与注册
- 自动发现网络中的 KVM Host 设备 (mDNS / 手动添加)
- 自动枚举本机 USB 采集卡 (`/dev/video*`)
- 支持 Host 类型: esphost (ESP32-S3), armhost (CM4), stmhost (F103)

### 2. 设备配对
- 将一个 Host 与一个采集卡绑定为一个"KVM 通道"
- 配对信息持久化到数据库
- 支持动态解绑 / 重新配对

### 3. 统一 Web 面板
- 多通道并排/切换视图
- 统一 HID 输入 → 路由到当前焦点通道的 Host
- 集中日志 / 状态监控
- 固件批量升级

### 4. 通信协议
- Host 设备: REST API + WebSocket (统一接口)
- 视频流: 本地 USB 采集卡 → MJPEG / H.264

## 技术栈 (暂定)

- **前端**: React / Vue
- **数据库**: SQLite (设备配对、日志)
- **部署**: Docker / systemd
